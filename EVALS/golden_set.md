# Golden Set

## ratchet
- Pin a clean workspace and list the newest ratchet ref.
- Mutate a tracked file, restore the ratchet ref, and confirm the file rolls back.

## spar
- Reviewer rejects a material gap and triggers exactly one fix round.
- Cosmetic-only findings are auto-approved by the non-material filter.
- Judge disagreement is surfaced instead of silently overridden.

## campaigns
- Start a campaign, append milestones, reopen the DB, and resume the open objective.
- Close a campaign and prune old completed rows.
