### 优化建议

1. **配置 Tushare**:
   ```bash
   echo '{"token": "your_token"}' > ~/.hermes/config/tushare.json
   ```

2. **配置 OpenAI**:
   ```bash
   echo '{"api_key": "sk-..."}' > ~/.hermes/config/openai.json
   ```

3. **或安装 Ollama** (本地免费):
   ```bash
   # 安装 Ollama
   curl -fsSL https://ollama.ai/install.sh | sh
   
   # 下载模型
   ollama pull qwen2.5:14b
   ```
