# Hermes Agent v0.13.67

## Android

- Adds saved and direct Tasker-style HTTP request automation for GET, HEAD,
  POST, PUT, PATCH, and DELETE over `http://` and `https://` URLs.
- Saves Tasker-compatible `%HTTPR` and `%HTTPD` variables after saved HTTP
  runs, with optional custom status/body variable names.
- Imports safe Tasker XML HTTP Get, HTTP Head, HTTP Post, and HTTP Request
  actions as disabled Hermes automation records by default.
