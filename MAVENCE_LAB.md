# Mavence Lab

## Mission Statement

**Mavence Lab advances the frontier of AI agent security and autonomous systems through rigorous security research, red teaming, and the development of secure-by-design AI agent architectures.**

---

## Focus Areas

1. **AI Agent Security** — Identifying and mitigating security vulnerabilities in autonomous AI agent systems, including prompt injection, tool abuse, and agent-to-agent attack vectors.

2. **Penetration Testing & Red Teaming** — Conducting structured security assessments through Hack the Box (HTB), Bugcrowd bug bounty programs, and live red team engagements to discover real-world vulnerabilities.

3. **Secure Agent Development** — Building security-first AI agent frameworks (like Hermes Agent) with built-in safeguards, command approval systems, and containerized execution environments.

4. **Security Automation** — Developing automated tooling for vulnerability scanning, security auditing, and continuous security monitoring of AI systems.

5. **Research & Knowledge Sharing** — Studying emerging AI security threats, publishing findings, and contributing to the security community through open-source tools and documented research.

---

## Core Activities

- **Bug Bounty Hunting**: Active participation in Bugcrowd and other platforms
- **HTB Labs**: Regular practice on Hack the Box for skill development
- **Agent Orchestration**: Building and securing multi-agent systems
- **Security Tooling**: Creating automation scripts, cron-based security jobs, and integrated security workflows
- **Code Review**: Security-focused code audits of AI agent implementations

---

*Founded on the principles of adversarial thinking, continuous learning, and building secure autonomous systems.*
