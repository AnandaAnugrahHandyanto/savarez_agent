## 최근 오르비 신호 요약 (2026-04-22 기준)

### 직접 확인한 신호
- 실시간 검색어 상위: `언매`, `월례`, `화작`, `세지`, `기하`, `국어`, `서울대`, `반수`, `n제`, `2028`
- 최근 핫/대표 글 정서:
  1. **시험끝낫는데친구없어서울엇어..** → 시험 뒤에 남는 공허감, 관계 결핍
  2. **중간 ㅈ망 / 반수열차 탑승권 구매 마렵네** → 성적 한 번에 자존심 붕괴, 즉시 반수 상상
  3. **n수할 때 뭔가 좀 편안했음** → 사회/연애/비교에서 도피하고 싶은 정서
  4. **6모 적백받으면 고백받아준다** → 연애조차 성적 조건화, 농담처럼 보이지만 절박한 로망

### 감성 결론
오르비형 로맨스는 그냥 달달하면 약하다. 먹히는 건 아래 조합이다.
- **입시 성적 = 사랑의 자격**처럼 느껴지는 잔혹한 조건
- 시험 직후의 공허함, 친구 없음, 혼자 귀가하는 감정
- 반수/재수 욕망과 연애 욕망이 충돌하는 상황
- 좋아하는 마음을 곧바로 말하지 못하고 **점수/입결/자존심**으로 우회하는 화법
- 혐관처럼 시작하지만 실은 서로의 패배감을 가장 정확히 알아보는 관계

### 이번 작품에 반영할 핵심 장치
- 여주가 남주에게 던지는 한 줄: **"6평 수학 백 받으면, 내가 먼저 고백할게."**
- 남주는 모욕처럼 받아들이고 독하게 공부함
- 여주도 사실 반수 고민 중이며, 그 한마디는 장난이 아니라 자기식 구조 신호였음
- 엔진은 로맨스지만 피부는 철저히 오르비식: 자습실, 더프, N제, 월례, 반수, 입결, 구겨진 프린트, 시험 끝난 후 편의점
