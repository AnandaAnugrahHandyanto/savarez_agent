# Character anchor — 한시윤

## Fixed visual identity
1. 창백한 피부의 마른 체형 남학생
2. 짙은 흑발, 부스스한 중간 길이 레이어드 헤어
3. 눈썹과 눈가를 덮는 긴 앞머리
4. 가늘고 긴 반쯤 처진 눈매 + 짙은 다크서클
5. 작고 날렵한 계란형/역삼각형 얼굴, 뾰족한 턱선
6. 얇은 입술의 무표정한 얼굴
7. 어두운 네이비/블랙 무지 맨투맨 또는 후드
8. 차갑고 우울한 분위기의 조용한 인상

## Drift to suppress
- 눈이 갑자기 커지거나 동그랗게 어려 보이는 방향
- 턱선이 과하게 매끈한 미남형으로 바뀌는 방향
- 목과 어깨가 갑자기 두꺼워지는 방향
- 앞머리 길이/정수리 볼륨이 컷마다 크게 흔들리는 방향
- 과한 젖은 머리, 과한 광택 피부, 아이돌형 스타일링
