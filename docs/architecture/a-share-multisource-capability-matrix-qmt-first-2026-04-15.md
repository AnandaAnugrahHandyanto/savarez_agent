# A股短线多源数据能力矩阵（QMT 主轴版，2026-04-15）

## 结论

当前最合理的数据架构不是“多个外部源拼日报”，而是：

> **以 QMT 实时数据作为主交易引擎，IFIND 负责板块/题材/事件确认，Tushare 负责日频情绪/资金/龙虎榜增强，AkShare / efinance / PyTDX 作为补洞与备用层。**

新的主次关系：

- **QMT：主轴 / 交易实时层**
- **IFIND：确认层 / 结构层**
- **Tushare：增强层 / 盘后层**
- **AkShare / efinance / PyTDX：补洞层 / 容灾层**

整体判断：

> **短线交易真正最值钱的是盘中承接、竞价、成交额、板位、炸板/回封与最强股切换，这些都应由 QMT 主导；其它源的价值是提高“认知正确率”和“盘后复盘质量”，而不是替代 QMT 的交易主链。**

## 推荐的系统分层

### Layer 1：QMT 主交易实时层

职责：

1. 生成盘前候选池
2. 读取竞价、盘口、承接、成交额、涨幅、板位等实时字段
3. 负责盘中升级/降级
4. 生成盘前、盘中、尾盘三段式输出

QMT 应主导：

- 竞价候选
- 可执行主攻 / 备选 / 观察 / 回避
- 盘中最强切换
- 候选升级/降级
- 炸板/回封行为
- 全日轨迹

### Layer 2：IFIND 确认层

职责：

1. 确认候选是否属于真实主线题材
2. 验证板块/行业/成分归属
3. 构建题材内龙头/次龙/跟风层级
4. 提供研报/公告/专题信息补充

IFIND 应主导：

- 题材成分确认
- 行业/概念归属
- 板块一致性
- 候选题材命中
- 题材层级图
- 事件型补证

### Layer 3：Tushare 增强层

职责：

1. 提供盘后情绪/资金/龙虎榜增强
2. 生成多日持续性与次日前瞻
3. 为 QMT 结果提供“市场背景与概率修正”

Tushare 应主导：

- 涨停/跌停/炸板/连板/最高板
- 个股/板块/北向资金
- 龙虎榜
- 题材持续性
- 解禁/预告/增减持/回购

### Layer 4：补洞与容灾层

- `AkShare`：宽表、榜单、资金流、概念、公告补洞
- `efinance`：东方财富系资金流和板块热度补洞
- `PyTDX`：分钟级和准实时行情补强
- `easyquotation`：轻量快照备用
- `Baostock`：历史基础数据底座

## 针对当前短线日报/盘中链路的最佳分工

### 盘前

- **QMT 主导**：竞价、盘口、成交额、初筛候选
- **IFIND 辅助**：题材真实性、板块一致性
- **Tushare 辅助**：昨日情绪反馈、题材持续性、昨日涨停/连板反馈

### 盘中

- **QMT 主导**：盘中最强、候选升级/降级、承接、炸板/回封、动作建议
- **IFIND 辅助**：题材内地位是否仍为核心
- **PyTDX 备用**：分钟级/分时补强

### 盘后

- **QMT + Tushare 主导**：全天最强演变 + 情绪/资金/龙虎榜 + 次日预案
- **IFIND 辅助**：板块画像、真实成分、事件确认

## P0 / P1 优先级

### P0
1. 以 QMT 为主轴重构 schema：把 QMT 字段设为主字段，外部源只做 enrichment
2. 在 QMT 结果中补齐盘中结构指标：
   - 炸板数
   - 回封数
   - 连板数
   - 首板数
   - 盘中最强切换次数
   - 候选升级/降级事件
3. 接 IFIND 做题材层级确认：
   - 题材真实成分
   - 候选在题材中的排位
   - 龙头/次龙/跟风层级
4. 接 Tushare 做情绪/资金/龙虎榜增强：
   - 炸板率
   - 连板率
   - 北向/主力资金
   - 龙虎榜活跃股

### P1
5. 做多时点持续性：QMT 盘中多快照 + Tushare 多日趋势 + IFIND 板块多日确认
6. 做次日前瞻事件层：研报/公告/解禁/预告/增减持
7. 视 QMT 粒度不足情况再接 PyTDX 做分钟级补强

---

## 当前最推荐的落地顺序

1. **先改 QMT 主轴 schema**
2. **补 QMT 盘中结构指标**
3. **补 IFIND 题材/板块确认接口接入点**
4. **补 Tushare 情绪/资金/龙虎榜**
5. **最后再接 AkShare / PyTDX 作为补洞**

---

## 最终判断

当前你最有优势的数据不是 IFIND，也不是 Tushare，而是：

> **QMT 实时候选池 + 盘口 + 多快照刷新链路。**

所以后续所有设计都应该围绕：

> **QMT 决定“能不能做、什么时候做、是否降级”；IFIND 和 Tushare 决定“为什么做、是不是主线、市场是否支持你做”。**
