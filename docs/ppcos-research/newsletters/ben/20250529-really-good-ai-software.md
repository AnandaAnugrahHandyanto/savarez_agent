I write a newsletter about startups and investing—for ai builders of all levels.
I record mini-tutorials, review tools I’m testing, share my insights from an exited founder turned investor.
a big few days of launches, and ofc i’ve been sick - but feeling better and i’ve got two vids today from two actually good ai tools;
first up, factory [ https://substack.com/redirect/0bbb7607-9855-4a18-abd9-f4c62bdb3839?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ]. it’s finally available for general access. sign up and use it now [ https://substack.com/redirect/0bbb7607-9855-4a18-abd9-f4c62bdb3839?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ].
here’s me using it to change a load of code across different files.
(also on youtube [ https://substack.com/redirect/56e4284d-12e1-457f-95b3-edf2f25276fc?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ])
so this is a background agent, it’ll write code for you, submit PRs, etc. it’s also the first platform that allows you to work with agents: local + synchronous and remote + asynchronous.
i’ve tried a lot of background agents; cursor, codex, jules, devin etc and factory did what i asked with zero problems (and zero lines of code written by me).
(disclaimer - i’m an investor, and a really fucking happy one with how well this works)
next up, retool agents. you can use it here [ https://substack.com/redirect/b0f13b8a-7250-4bfe-bcaa-08ec5010408c?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ].
retool was around in the no-code days to help teams create internal tools but it was more low-code and less no code tbh. BUT NOW….
they’ve released their ‘Agents’, which is a super simple agent builder with a lot of powerful features; mcps, workflows, trigger via email or chat, evals, datasets and logs.
these are all the tools devs have available today but now non-devs can use them too!
here’s me building a couple of agents in less than 10 mins.
(also on youtube [ https://substack.com/redirect/7f1ead24-9500-4e0d-9ea8-46e14718df89?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ])
this is genuinely one of the simplest ways i’ve tried to create agents - particularly ones that aren’t basic ones. Like my founder analysis tool is just an agent with an interface now (or trigger by email).
retool benefits from having years of building a lot of components for workflows over the years, and now their AI system has access to those.
the ai assistant helps craft instructions, creates its own tools you can use and customise.
i truly recommend this to build agents [ https://substack.com/redirect/b0f13b8a-7250-4bfe-bcaa-08ec5010408c?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ], and share them with the community [ https://substack.com/redirect/77f65b28-e2b4-4f4e-8ae6-01e94fbfe639?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ] to inspire one another and remix interesting ones!
