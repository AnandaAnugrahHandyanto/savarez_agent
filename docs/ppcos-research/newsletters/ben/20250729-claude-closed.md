The all you can eat buffet is ending. Claude Code is getting weekly rate limits [ https://substack.com/redirect/cab965af-0c4b-465e-bda4-bb8bd726d588?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ] that Anthropic says will impact only 5% of the users (i.e. us). I saw this cool tool: Claude Count [ https://substack.com/redirect/f25141be-2d21-412e-bc43-ba946ae2ae35?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ] which shows a leaderboard of how much you’re using Claude Code. I made it to the top 20!
Claude Code also has subagents [ https://substack.com/redirect/36e47b50-5e30-4d7d-b875-b7e617351caf?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ] -  You can create these within the Claude Code interface, and Claude can write them for you. This basically gives them a set of instructions, things to do, and tools they can use, and it's going even more in the direction of apps becoming agents now. And some inspiration [ https://substack.com/redirect/2b3b3dbd-edcd-4b71-a8ff-ceaf067c28f3?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ] for what subagents to create.

Check out this interview with some of its makers [ https://substack.com/redirect/4ddd1199-a8b3-4fb0-a5a1-d687fbcef9df?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ].
Runway has a new video model - Aelph [ https://substack.com/redirect/3c8673dd-37c7-4939-925c-92a5e59df05f?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ], made primarily for editing videos like creating a new angle of a scene, adding/removing objects, and transferring style from one clip to another. Related: Netflix’s co-CEO mentioned the use of generative AI in one of the sequences in an Argentinian show on the platform, so yes, AI video is staying.
Z AI, another Chinese company released a big open-source model that beats Opus on benchmarks. GLM-4.5 and 4.5 Air [ https://substack.com/redirect/e0571c4f-d907-4b9c-90b4-47a433243f4e?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ] are two variants of their new model, both very cheap and with near frontier performance, especially when it comes to tool calls.
