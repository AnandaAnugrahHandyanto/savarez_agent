Agentic browsers like Perplexity’s Comet remain vulnerable to indirect prompt injection attacks [ https://substack.com/redirect/0fd9d184-20cb-4b9f-a11d-21b84cb8a0a8?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ]. Which is why folks like Browserbase push for “1. your agent should have it's own browser with limited access to it's information, and 2. it should run far away from your data.” - this thread [ https://substack.com/redirect/dd2842d5-6e78-4ccc-b2b3-c24daba7435c?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ] highlights more of why it’s a massive issue.

It’s like a backend in a box (which is something I’ve been posting about a lot).
