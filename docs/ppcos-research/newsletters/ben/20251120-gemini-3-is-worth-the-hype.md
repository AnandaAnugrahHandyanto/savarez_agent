The newsletter for the technically curious. Updates, tool reviews, and lay of the land from an exited founder turned investor and forever tinkerer.
Hey folks,
Gemini 3 Pro is out [ https://substack.com/redirect/45d4d5d2-10b1-4af4-8d10-f5754e0f1242?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ] beating GPT-5.1, Sonnet 4.5 on all benchmarks but one (SWE-Bench Verified). It’s only slightly smarter than other models, but it is much better at vision. It scores 72.7% on a benchmark for screenshot understanding; the second best is 36.2%. Faster and slightly expensive [ https://substack.com/redirect/9406b413-cc52-4edd-bcb5-bb17d91c6ed1?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ] than 2.5 Pro.
My vibe check: it’s a good model, it’s eager to commit changes (!!), isn’t as good as following instructions as Codex (maybe some new prompting quirks needed?) and definitely [ https://substack.com/redirect/9220beda-2d56-47f0-9a17-cb21432ad907?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ] good at frontend.
This launch comes with a new IDE from Google: Antigravity [ https://substack.com/redirect/fd293741-7d06-4ad4-9489-79be6ceae65c?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ] (courtesy of acquiring the Windsurf founders). It also has an Agent Manager and a browser for the agent to see/test what it has built. I tried it for a few hours: Tab complete feels very slow, and the agent is over-eager to implement a plan (while in planning mode). The Agent Manager treats extra documents like plans, task lists as separate “artifacts” and doesn’t clutter your codebase. I liked that. It took me a while to get the browser integration working, but once it was set up, it was really nice and fast too vs Atlas.
In other stuff [ https://substack.com/redirect/cac85168-fad7-44f9-8d38-8281876665f5?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ], they released Gemini Agent (for ultra subs only), introduced Dynamic UI in chat, teased Gemini 3 Deep Think [ https://substack.com/redirect/3e3788dd-a7a4-4269-9aec-388c63c4dea2?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ] and rolled it out in Search (via AI mode [ https://substack.com/redirect/26018751-05e6-44aa-9fdf-f89379bc3baa?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ]) on day 1.
They are not done (potentially nano banana 2 today), but neither are OpenAI and xAI. OpenAI released two models: GPT-5.1 Pro [ https://substack.com/redirect/0189c828-db44-4ffd-bf9e-63c37a483937?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ] and GPT-5.1-Codex-Max [ https://substack.com/redirect/d3c919b6-c214-45a7-bfb1-2b6049bc8f2e?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ] to follow up on Gemini 3, and xAI has released Grok 4.1 Fast [ https://substack.com/redirect/ef014ab8-5307-4639-b488-70dc3c007b8d?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ]. These three models all look like they’d do better than Gemini 3 Pro, but only on specific tasks (hard academic problems, code generation and tool calls).
An underrated release from Meta: SAM 3 [ https://substack.com/redirect/bc2132ee-5ca6-4155-8e82-b34c754fbc93?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ]. SAM (Segment Anything Model) family of models can take an image/video and create an overlay of any individual or group of objects in it. Meta is partnering with Roboflow [ https://substack.com/redirect/96f2d94f-c008-4ada-b513-05f7895eff3a?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ] to let people fine-tune SAM on their use cases, and it’ll use SAM in Instagram’s video editing app called “Edits”.
One API for All Your Voice AI Workflows. Stop wasting time juggling voice AI vendors. AssemblyAI combines multilingual Speech-to-Text, speaker diarization, speech understanding & LLMs in one developer-friendly API. Trusted by Granola, Dovetail & Ashby. Free to try, pay-as-you-go. Start building voice AI today [ https://substack.com/redirect/d55b2319-0157-41d1-a727-84a9ae35a921?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ].*
I’ve been talking with TELUS (one of Canada’s largest telecom companies) this year. They built a platform that allows 70k+ employees pick from over 30 LLMs to build copilots. I chatted with them and wrote about their story here [ https://substack.com/redirect/b71f20e4-0c44-4f44-b5ec-3caa31a124a9?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ].
