Droid can now scan your repo and tell you if it is ready for agentic software development [ https://substack.com/redirect/79b77a67-1836-4819-96f8-277fd75b5d1b?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ]. You can run it in a droid session with /readiness-report directly in your terminal, or view readiness across your organisation in the app. There’s also an option to access reports programmatically via API. It also suggests quick and longer-term fixes so that you can compound the work you can get done.
Claude has a new constitution [ https://substack.com/redirect/0bafa74b-1bf5-4eef-9021-a55b40dc9304?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ], and it has about 22.5k words (up from the old 2.5k words). Anthropic uses this document to guide Claude’s behaviour during training as well as when providing it to users. The new constitution has 5 major sections: being helpful, following Anthropic’s guidelines, being broadly ethical, being broadly safe and notes on Claude’s nature.
