Hey folks,
I haven’t done a list of tools that I use regularly (aka my daily-drivers 😉) so here’s some that I switched to and have not churned from:
Factory.ai [ https://substack.com/redirect/e3d9583e-b270-4f2e-9108-83c0b11e1f59?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ] - browser-based AI coding tool. Super easy to use and the actual agent is better than Claude Code and others (ie it doesn’t just list files and search patterns). I used it this morning to build a timezone converter that works by highlighting text (here’s it in action [ https://substack.com/redirect/55f5445d-0649-4554-8ff1-91517e465da7?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ]). oh and some other BIG news coming here v soon.
Granola [ https://substack.com/redirect/7473d3dc-3300-46cf-a207-f9c6d4e27932?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ] - it’s just the best meeting note taker. Not being present in the call is just better for everyone. They have a Zapier integration to pull out notes but waiting for their API/MCP so I can get factory to make some magic workflows and save to my folders. (linked my referral link but I dont get money, just pride)
Dia Browser [ https://substack.com/redirect/96bd5c4a-b79b-4831-af16-4b380d8c8879?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ] - having a quick AI assistant accessible on any web page or when opening a new tab has become a very go-to move for me. And I love the custom skills for quick workflows (like pull all the AI news from hacker news today). 5 free invites here [ https://substack.com/redirect/20d80c9c-d33c-4982-bc24-c96c2c17a534?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ]
ChatGPT/Claude - mostly only using these on mobile atm to be honest. I flit between the two, no loyalties or ties to either tbh.
Monologue [ https://substack.com/redirect/9e82bbb0-e378-419e-97b0-a0dc3b4c48fb?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ] - this replaced Whispr Flow for me; speech-to-text transcription when on my laptop. (linked my referral link but again, no money, just dopamine)
And honestly, that’s pretty much it! I try a bunch of email tools but nothing is perfect, so I’m going to just use Factory to make my own (I’ve built A LOT with it recently - more actual shipped projects than previously using Claude Code). 🫨
OpenAI released a new speech-to-speech model called gpt-realtime [ https://substack.com/redirect/2c8b3291-931d-4023-90e3-866b94d1fe4a?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ] for developers. Plus, the Realtime API now supports MCP and image inputs. Essentially, you now have the toolkit for building long audio/video chat sessions. Pricing’s not there though, gpt-relatime is 20% cheaper than gpt-4o’s realtime pricing, but it’s still too expensive to be packaged into a $20/mo plan. It is $32/$64 input/output per million “audio” tokens, that’s roughly 15 minutes of chatting per $1 of API spend.
xAI has a new coding model: Grok Code Fast 1 [ https://substack.com/redirect/0c3b8bd4-7fdc-4883-89a2-ead142cf63e4?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ]. Unusual choice of optimisation (faster vs smarter), but users in Cline, Cursor and other tools are liking the model. I haven’t tried it yet.
After years of not training on users’ data by default [ https://substack.com/redirect/18388947-3b2f-46ce-93dd-394ff7e95ea4?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ], Anthropic will now start doing that from 28th Sept. Of course, you can opt out, but another one of Anthropic’s "oh we don’t do that—like other evil labs” is down the drain.
a16z releases its consumer apps data every 6 months, and Olivia found a spike in Claude’s traffic that she couldn’t explain. Keshav went through Ben’s Bites editions from that week and found a likely reason [ https://substack.com/redirect/ab0ca7d1-6947-4c53-b4ea-ef4a36c6dff0?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ].
Attio [ https://substack.com/redirect/3c30f7e9-4bd6-488b-b07c-d234be33673a?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ] is the AI-native CRM for the next generation of teams.
