On to the news!
People are just realizing that “Harness is the new app [ https://substack.com/redirect/a7c12406-c3ab-4cbe-92fd-cfa0edf575f5?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ]” but this was very clear early this year when we saw Claude and Gemini play pokemon. We’ve been talking about [ https://substack.com/redirect/af1ba1ab-720c-46ef-896a-767373a268ee?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ] it since May but it’s gaining even more momentum now. Cursor is talking about improving their harness for Codex models [ https://substack.com/redirect/6dd8ce76-029c-4829-9a6b-6db1ee22f053?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ] and Poetic used a custom harness Gemini 3 Pro to get the best score on ARC-AGI 2 [ https://substack.com/redirect/53757840-cf79-46e4-b286-cc71d29e5086?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ].
Google upgraded Deep Think with Gemini 3 [ https://substack.com/redirect/462d1fe6-140e-49ef-b4c5-c5220f77f118?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ] for Ultra plan users ($250/mo). Shopify’s CTO and prev. Head of AI at Microsoft, Mikhail Parakhin [ https://x.com/MParakhin/status/1997392115777638545 ], thinks the new version is better for coding something from scratch and notices more corner cases (and it’s faster), but still has some rough edges.
You can now delegate tasks to Claude Code [ https://x.com/claudeai/status/1998109170449719647 ] directly from Slack. I played around with Claude Code on the desktop app and it’s a nice interface for Claude as an Agent. I won’t be surprised if Claude app starts to look like that by default in 2026.
Folks at HuggingFace got Claude Code and other CLIs to fine-tune an open model [ https://huggingface.co/blog/hf-skills-training ]. It’s available as a skill. Here’s an idea for you to steal: make a local email processing app with a 3-7B model.
