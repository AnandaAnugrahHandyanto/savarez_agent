🔎 News worth knowing
OpenAI launched o3 Pro [ https://substack.com/redirect/1d1f18eb-22fd-4ec3-a107-6bbd3900a3c5?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ] for Pro and (with limits) Teams plans. o3 pro feels much  better than o3, but the thinking time for each query makes it an annoying model to use. I’m getting 10-15 mins of thinking time per query on average, and when the code doesn’t work after 20 mins of scrolling twitter, it’s frustrating. Sam Altman also confirmed the 80% price cut for o3 [ https://substack.com/redirect/d061a436-da46-499e-a24c-99fbf9ead233?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ], making it now cheaper than GPT-4o in API (great time to be a builder).
