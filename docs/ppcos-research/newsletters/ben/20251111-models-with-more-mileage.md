GPT-5-Codex has a mini variant now. [ https://substack.com/redirect/375640cf-7a23-4245-a48c-3a42dd306d78?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ] It’s not that far from the big brother on SWE-Bench Verified. The model is only available in Codex CLI and IDE extensions. API is coming soon, though. Codex-mini gives you 4x the mileage of Codex. Also, all paid ChatGPT plans now have double the rate limits for Codex.
Kimi K2 Thinking [ https://substack.com/redirect/2da458bf-6e25-493d-aaae-9ae4e24a0495?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ] is an open-weights model that is now #1 on many benchmarks like HLE & BrowseComp across all models (not just open ones). Kimi models seem to have really great writing taste, while being great at coding tasks and being cheaper.
So the trend of models that go further continues with two options: closed-source RL-ed models (Composer, SWE-1.5) and huge MoE open models (GLM, MimiMax, etc.)
Replit now lets you add AI features to your apps [ https://substack.com/redirect/615db84f-90e8-4e92-aeb6-9c8e3f75dcb7?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ] with zero extra setup. No need to make developer accounts, get API keys, etc. You can use 300+ models, including Gemini, OpenAI, Claude or even open-weights models from OpenRouter. The AI cost is billed against your credits. Smart move tbh.
There’s a new stealth model on OpenRouter named Polaris Alpha, which looks a lot like GPT-5.1 [ https://substack.com/redirect/9e18c8a5-3c41-4ef1-84de-3c7275148411?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ].
