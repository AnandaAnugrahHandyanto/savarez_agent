Also, Meta acquired PlayAI a couple of days back. A wild few days in acquisition land.
Another Chinese lab, Moonshot AI is taking over the open-model scene with Kimi K2. It’s a massive 1T MoE model with no reasoning yet (tech blog [ https://substack.com/redirect/d6d68672-78a6-4a18-afb3-ddadbca2057e?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ]), and it easily beats Claude Sonnet 4 and pokes Opus 4 too. It’s great at coding and tool calling, but that’s not it. On this creative writing eval [ https://substack.com/redirect/fc4bba7b-b5df-430b-9e46-32c1bb3da6af?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ], it just blows everything out of the water, avoiding most of the telltale signs of AI writing. This is what Llama 4 Behemoth was supposed to be.
You can use it on the web [ https://substack.com/redirect/21a21001-205e-43dc-92c5-84bc4ee77327?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ], it also has an API [ https://substack.com/redirect/d1c32d8c-e8f3-423e-9f7e-f26ec001bbb0?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ] (makes it possible to run in Claude Code [ https://substack.com/redirect/1786fb50-c247-40e7-b3ff-40ca3322cdad?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ]), but that’s slow, and the folks at Groq are already serving it at superfast speeds [ https://substack.com/redirect/d7804086-648b-4f3c-86fa-f462256431f8?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ].
Grok has a new feature: Companions [ https://substack.com/redirect/0fcaff32-f8dd-4aca-872e-069697f5b5bd?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ]. It’s built in partnership with Anichat. Grok powers the dialogue (with voices), and Anichat covers the generative animation. One of the avatars Rudi is a red panda, who can be good and bad, and the other one is Ani, a borderline nsfw avatar based on an anime character. That company Meta just acquired used to clone AI voices, so expect something like this from Meta too
Claude finally has a directory of pre-built MCP connectors [ https://substack.com/redirect/a9448bcb-0557-45c5-a0d1-3a86d0d8bbae?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ] to connect to Notion, Figma, Stripe and more. Plus this is also live for the Pro plan ($20/mo) and web app. The desktop app has some special ones like allowing Claude to use Chrome, read your files, fill PDF forms etc. You can also add a custom connector if you have a remote MCP url—like from Smithery [ https://substack.com/redirect/6bf9a6fc-13e6-4d2f-84fe-c5bc797a3f7a?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ] which is a portfolio company I use several times a week.
