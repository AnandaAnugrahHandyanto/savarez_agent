Hey folks,
I just posted a little video walkthrough [ https://substack.com/redirect/434ee660-e8eb-4b32-97a6-3fdc9d5461ba?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ] of something I built! I gave a tweet of a video to droid [ https://substack.com/redirect/5494150b-6cd3-4f52-ba35-cd9f80186764?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ], said reverse engineer this and rebuild it (and it did!) It lets you have a little dashboard for agents looping on tasks (or you can use GitHub issues). Super fun little project to test out how easy it is to point at a video and reverse engineer something in a morning. Enjoy!
Cursor let agents build a web browser from scratch [ https://substack.com/redirect/53bd4ad6-9505-42f6-9518-2e0588730869?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ] by letting them run for about a week. They found GPT-5.2 to be the best model for longer, more autonomous work (vs. Opus 4.5) and prompts to matter more than they expected. Also, OpenAI just made GPT-5.2-Codex [ https://substack.com/redirect/70015aa8-1968-437e-8774-c7fe29df236e?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ] available in the API, i.e. it’s available in Droid, Cursor, Windsurf and other apps too.
Claude Code can now search the right tool [ https://substack.com/redirect/1168ef94-baae-47b1-b571-63b3bee765d9?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ] from your MCP server without clogging up all of your context window. Also, you can now add a comment by pressing tab when accepting/rejecting a permission prompt [ https://substack.com/redirect/6f716a5e-f53b-42a9-b740-5ea60e366661?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ], i.e. “yes, and {do it this way}.”
Anthropic is expanding its experiments garage. Anthropic’s Labs team [ https://substack.com/redirect/9c1d6f4a-aa30-44a3-bb03-3d239b538d56?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ] is hiring builders to create new products like Claude Code, MCP, and Claude in Chrome under Mike Krieger and Ben Mann.
Google is adding Personal Intelligence [ https://substack.com/redirect/feffa33f-2812-43ab-9d42-023ee2c5104e?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ] i.e. deep data integration with your Gmail, Photos, YouTube history and more into the Gemini app. I tried testing it, but it’s US only, and I’m very suspicious of it working as well as the demos show.
I’ll be joining Every for their Vibe Code Camp [ https://substack.com/redirect/e288d7e7-ff5f-4753-9402-572b65c4f02b?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ] - where a bunch of us ‘vibe-coders’ will be building stuff, sharing what we did and tips. Other folks you know will be there!
