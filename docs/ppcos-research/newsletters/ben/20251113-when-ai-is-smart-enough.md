Hey folks,
GPT-5.1 is out [ https://substack.com/redirect/a3282945-f2fc-4c68-81ae-b56bb9694edd?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ]—only in ChatGPT for now, API access is coming soon. We get two new models: GPT-5.1-Thinking and GPT-5.1-Instant (which can also think if needed, but just a little). The focus is on two things: vibes and safety, not smartness. OpenAI claims it has better vibes than 4o when chatting, and it’s safer than GPT-5. ChatGPT is also getting new default personalities that you can pick from. Here’s how they all respond [ https://substack.com/redirect/24b70a23-3579-4835-bea1-0d04a5d9317f?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ] to the same question [ https://substack.com/redirect/314f3e61-7929-49c5-b2f5-7d842e9fec48?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ].
This year, we have seen a bunch of research previews that generate navigational environments from images/prompts. Marble by World Labs [ https://substack.com/redirect/63ec5db5-d616-4486-acdb-ecb5b15ea0cf?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ] is another one on the list. It’s not interactive like some others, but it’s available to use as a product, and I’m really impressed by the quality.
Realistic video was one of the key trends in 2025 (a noticable improvement from 2024’s “still looks weird/fake” videos). We’ll see a lot of these realistic/interactive worlds next year with likely a lot more interest in VR headsets—Samsung just launched one couple weeks ago, Steam is also launching one for games next year.
