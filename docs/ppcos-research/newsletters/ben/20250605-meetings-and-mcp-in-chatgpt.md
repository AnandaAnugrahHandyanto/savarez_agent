🔎 News worth knowing
ChatGPT is finally getting MCP connectors [ https://substack.com/redirect/61ab48de-7924-44aa-9c95-0adecbda6764?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ] → You can use these (and other pre-built connectors) in Deep Research mode. The pre-built connectors [ https://substack.com/redirect/38c5b129-5d38-4ac2-921e-434353297604?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ] are available in the cheapest Plus plan ($20/mo), but custom connectors via MCP are not on it yet.
ChatGPT Record Mode [ https://substack.com/redirect/1b5dc0cb-cb79-459a-8649-678782b9aa98?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ] → On the ChatGPT MacOS desktop app, you can record your meetings and get a) full transcript and b) a pointwise detailed summary with citations to your transcript. ChatGPT can also refer to these transcripts for further tasks in new chats (if you want to). This is available only on the Team plan for now.
Codex, the SWE agent by ChatGPT [ https://substack.com/redirect/8cb0f2b4-9744-4f7b-a962-1214f59729dc?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ], is now available on the Plus plan with some upgrades. The most important one: you can now enable internet access for Codex, i.e. Codex can install extra dependencies while working. I tried working with it yesterday, and the process is now much smoother.
In SaaS, NDR separated the best from the rest. In GenAI, it’s the Emergence Rate [ https://substack.com/redirect/72ffd4f3-b851-4aa6-b7e7-63cc2b970fac?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ]. This new framework from Emergence Capital tracks whether your AI product compounds value with every user interaction. Don’t scale static software. Build the perpetual algorithm.*
Mistral also released their coding assistant “Mistral Code [ https://substack.com/redirect/52855cbd-736b-4cd6-a066-f286ebccfa5f?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ]”, which looks more like GitHub Copilot than Claude Code. Also, Claude Code [ https://substack.com/redirect/6425d01d-8b3e-451f-b1ed-81530ea73f11?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ] is now available on the Claude Pro plan ($20/mo) with other features like Research and MCP Integrations [ https://substack.com/redirect/a93b3e28-96df-4d56-ba4f-5a407222e7d7?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ].
