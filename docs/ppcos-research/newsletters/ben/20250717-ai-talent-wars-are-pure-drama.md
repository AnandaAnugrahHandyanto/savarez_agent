ChatGPT’s image creation is one of the best, but it’s not really great when you want to do precise edits to an existing image. OpenAI just added high-fidelity editing to their Image API [ https://substack.com/redirect/ee8868cd-5242-4095-b740-775547f9a3c8?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ]. Editing faces, logos, and other details will now result in fewer instances of irrelevant changes.
Google is finally making 2.5 Pro available on AI mode in search [ https://substack.com/redirect/88595855-3360-431a-b0db-a9dc21748aba?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ] (+ deep research too). It’s still under a dropdown, not the default. And if you’re a normal google search user, it still takes 3+ clicks after you’ve hit enter on your query to reach ai mode. And search can make a call on your behalf to local business but i’ll trust that one when I try it - been watching that demo for half a decade now 😅.
The leaders of Claude Code left Anthropic a couple weeks ago for Cursor, but Anthropic has played Uno reverse. Both are back at Anthropic [ https://substack.com/redirect/c1cc5753-c2b5-4b11-be05-8d5c9be7681c?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ]. Claude Code supremacy continues. And feels like Anthropic is doubling down on features for Claude Code. Organisations that use Claude Code can now access detailed analytics [ https://substack.com/redirect/dec9a393-22cf-498d-a83b-f8e4f698cec6?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ] of their org wide use.
I’m tinkering with Claude Code interface for non-technical people so you don’t have to use it through the terminal (because Claude Code is just generally the best agent I’ve used, not just for coding tasks), and a few other UX/UI abstractions to make it easy. Will open-source when it’s in a reasonable state (bit rough around the edges atm but works!)
Voxtral [ https://substack.com/redirect/9c4edafe-5010-4023-9cc7-21afc432a67d?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ], a new family of voice transcription models from Mixtral, is worth paying attention to. It comes in two variants 3B and 24B—the 3B one is nice and small for running on your laptop and gives you roughly similar performance on audio transcription as Gemini 2.5 Flash. The 24B variant is good for self-hosting for companies because latency and cost will be much better than using a closed-source provider.
Hostinger Horizons is doing a live session on how to turn idea into apps without code/with AI for Ben’s Bites readers. Next Thursday, 24th July. Register here [ https://substack.com/redirect/0552e801-9bf3-4f3f-8d55-c16a8ef386a0?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ].
A quick shoutout to AI Tinkerers [ https://substack.com/redirect/dda398d7-0a3a-4c90-ad3c-06af537fec39?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ] - I went to the LDN event yesterday and met some great people. There were some awesome demos too, my favourite was personalised Hacker News [ https://substack.com/redirect/3bbbc3fe-5bb7-4def-9d12-c159930bfd74?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ] - you rate 10 random posts to onboard (swipe or click) then the feed shows you tags on what you’ll love, may like, and want to skip. You can see the prompt they generate for you and tweak it too if you’d like. I replaced my pinned HN with this one 😃. Josh built an MCP with auth and payments in 10 minutes. And I spoke to the founder of Superoptix [ https://substack.com/redirect/03136c10-3d31-4c4e-a592-51926d1bb061?j=eyJ1IjoicXd0ciJ9.0tEpqnOrcfuKYRYnEOurmlOZ_pVIUnS06konjQA-bCs ] - an agentic ai framework that automates the harder parts (ie, not the agent set up bit).
