## Summary

The hosts discuss Gemini 3 Flash as a game-changing model that combines near-Pro-level intelligence with Flash-tier speed and pricing (50 cents/million input, $3/million output). They compare GPT Image 1.5 vs Nano Banana Pro (Nano wins decisively for infographics and character consistency), explore the Firecrawl Agent MCP for reliable web research, and preview Gemini Deep Research Agent. The episode concludes with their 2025 model reviews: Opus 4.5 as best model despite occasional inconsistencies, skills as the emerging paradigm over MCPs, and frustration with GPT 5.2's unpredictable behavior.

## Key Insights

### Gemini 3 Flash
- **Pricing:** 50 cents per million input tokens, $3 per million output - up from 2.5 Flash but still cheap
- **Performance:** Hard to distinguish from Gemini 2.5 Pro level intelligence; feels like "a sped up, cheaper 2.5 Pro"
- **Tool calling:** Far better than Gemini 3 Pro - more precise, doesn't try to call 100 tools at once
- **Use case:** Great as a "recovery model" when Opus 4.5 or GPT 5.2 fails; solved tedious front-end bugs that Opus couldn't
- **Output quality:** Tight, accurate output that sticks to the brief; actually preferred Flash over Pro in some tests
- **Reliability:** More consistent than the current crop of frontier models (Opus 4.5, GPT 5.2, Gemini 3 Pro all described as "unreliable")

### GPT Image 1.5 vs Nano Banana Pro
- Nano Banana Pro wins on: infographics, character consistency, text accuracy, upscaling
- GPT Image 1.5 feels "reactionary" - good but not as reliable
- Both platforms have relaxed safety controls significantly

### Firecrawl Agent MCP
- Phenomenal for research - crawls multiple pages, makes decisions, verifies data
- Far more reliable than simple web search + paste approach
- Finally trustworthy enough to delegate research entirely to an agent

### Skills vs MCPs
- Skills = detailed prompts with context, procedures, brand guidelines baked in
- MCPs = action/retrieval focused tools (web crawl, database queries, APIs)
- "Skills with MCPs" = the winning combination for enterprise
- Skills allow team-wide sharing of business knowledge and processes
- Expect skills to become even more prominent in 2026

### 2025 Model Year in Review
- **Best Model:** Claude Opus 4.5 (with Sonnet 4.5 as runner-up)
- **Worst Model:** GPT 5.2 ("completely wild", "off with the fairies")
- **Observation:** No reliable go-to model right now - all frontier models feel "beta"
- **Gemini 2.5 Pro:** Was the workhorse but got "destroyed" before 3.0 release
- **2026 Prediction:** The year skills actually become useful; agent orchestration improves
