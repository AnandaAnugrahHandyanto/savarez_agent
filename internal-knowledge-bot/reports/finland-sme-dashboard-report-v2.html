How to sell this

  • Lead with one operations KPI, not AI features.
  • Offer 14-day pilot with clear success metric.
  • Demo dashboard modules: triage, escalation, KPI wall.
  • Bundle governance defaults for trust.
  • Land one workflow first, then expand.

Scenarios

  • Conservative (2 closes): €28,467 setup + €5,587 MRR
  • Base (4 closes): €56,933 setup + €11,173 MRR
  • Aggressive (6 closes): €85,400 setup + €16,760 MRR

Recommended stack

  • FI/EN/SV support
  • RAG + role/group controls
  • Human handoff + SLA queue
  • Audit + KPI analytics
