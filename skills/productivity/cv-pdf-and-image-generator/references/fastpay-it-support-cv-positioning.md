# FastPay IT Support / Linux Admin CV positioning notes

Use when tailoring Hevar's CV or application text for FastPay-style IT Support, Linux System Administrator, or infrastructure-entry roles.

## Public job-post signals observed

FastPay IT Support role emphasized:
- Desktop/laptop/printer/peripheral support
- Windows/Linux troubleshooting
- LAN/WAN, routers, switches, Wi-Fi
- TCP/IP, DNS, DHCP
- User accounts, permissions, access control such as Active Directory or similar
- Routine updates, backups, security checks
- Helpdesk requests, employee support, documentation
- Minimum ~2 years professional PC/network support experience preferred/required
- Certifications like CompTIA A+, Network+, Microsoft certs as pluses

FastPay Linux/System/Admin/DevOps-style posts emphasized:
- Linux servers, systemd, apt, permissions, users/groups, SSH, logs, disk/process monitoring
- Networking/firewalls/storage/security hardening
- Docker, Kubernetes basics, Bash/Python scripting, Ansible, monitoring/alerts
- Nginx, Git, CI/CD basics
- Java/Spring infrastructure, MySQL, Redis, MongoDB, Harbor, Vault, Prometheus/Grafana, Graylog, MinIO, on-prem Linux servers

## Positioning guidance

For IT Support applications, lead with:
> CS student with hands-on freelance IT support experience across Windows/Linux troubleshooting, OS installation, Wi-Fi/router issues, user support, Microsoft Office/Outlook, and basic networking including TCP/IP, DNS, and DHCP. Comfortable supporting employees/clients in Kurdish, Arabic, and English, documenting fixes, and learning infrastructure operations toward Linux/System Administration and DevOps.

Keep DevOps as a growth direction and proof of technical initiative, not the main identity for IT Support. The main identity should be:
- Technical Support
- IT Support
- Help Desk
- Troubleshooting
- Network/user support

## Interview story prep

Prepare concise practical answers for:
1. One website does not open: DNS, cache/browser, ping, nslookup, firewall, different network.
2. Printer not working: power/network, queue, driver, default printer, IP, test page.
3. PC slow: Task Manager, startup apps, disk/RAM usage, malware, updates, free storage.
4. User cannot log in: password, account lock, keyboard language, network/domain, permissions/reset process.
5. Wi-Fi connected but no internet: IP, gateway, DNS, DHCP, router, other devices, ping gateway/8.8.8.8/domain.

## Honest chance framing

For FastPay-style IT Support, a CV like Hevar's can be a realistic junior/support candidate if he explains freelance/practical work clearly. For Linux System Administrator, treat it as a training roadmap unless he has production server experience.
