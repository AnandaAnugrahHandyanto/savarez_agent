# Present Skill

Generate a PowerPoint (.pptx) slide deck on any topic the user specifies.

## Invocation

```
/present <topic> [--slides N]
```

- `--slides N` is optional (default: 5 slides)
- Topic can be anything: technical concept, project overview, poker strategy, etc.

## Workflow

1. **Pick a topic** from the user's request
2. **Choose slide count**: Default 5-7 slides. User can specify `--slides N`
3. **Pick a color palette** based on topic keywords:
   - Poker/card games → Forest Green (`1D6F42` primary, `2ECC71` secondary)
   - Tech/software/AI → Midnight Navy (`1E2761` / `CADCFC`)
   - Business/startup → Warm Terracotta (`B85042` / `E7E8D1`)
   - Finance/investing → Ocean Blue (`0F4C75` / `3282B8`)
   - Default → Charcoal (`2C3E50` / `34495E`)
4. **Use pptxgenjs** to generate the deck — write a Node.js script and run it

## Generating the PPTX

```javascript
const pptxgen = require('pptxgenjs');
// Set NODE_PATH for global modules:
const NODE_PATH = require('child_process').execSync('npm root -g').toString().trim();
```

Save the output to `~/.hermes/slides/<topic_slug>.pptx`.

## Slide Structure

- **Slide 1**: Title slide — dark background (palette primary), large title, subtitle "Slide Deck — Generated by Nyx"
- **Slides 2-N**: Content slides — light background (palette bg), white card with shadow, section title, 3 bullet points each
- Each content slide: left accent bar, slide number bubble, card with top accent border

## Design Guidelines

- Fonts: Arial Black (titles 44pt), Arial (section headers 20pt), Calibri (body 15pt)
- Shadow factory: `() => ({ type: 'outer', color: '000000', blur: 6, offset: 2, angle: 135, opacity: 0.12 })`
- **NEVER reuse shadow objects** — always call the factory function per element
- **NEVER use `#` in hex colors** (causes file corruption)
- **NEVER use unicode bullets** — use `bullet: true`
- Use `breakLine: true` between bullet items in arrays

## QA

After generating, verify the file exists and is > 10KB.

## Output

Return the file path: `~/.hermes/slides/<filename.pptx>`
