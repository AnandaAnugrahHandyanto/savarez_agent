# AI promos and small-budget provider advice — 2026-05 session notes

These notes came from a budget-choice conversation where the user had about $10/month and was comparing OpenRouter credit against a discounted Google AI Pro offer.

## Durable reasoning pattern

- For **Hermes/Mizuki power**, prefer **API/model credit** over a consumer app subscription.
- For **personal app workflows**, a subscription can be worth more than API credit.
- If the user only says "I use AI" and wants agent capability, treat OpenRouter/API credit as the default unless they name app-only features they will use.
- Always distinguish: app access, API access, IDE coding assistant access, storage/workspace perks, and student/free benefits.

## Google AI Pro promo observed in session

The user's local offer text listed:

- Google AI Pro at **IQD 13,100/month for 2 months**, then IQD 26,200/month after.
- **5 TB storage**.
- Gemini with 4x higher usage limits.
- Access to a Pro Gemini model for reasoning/math/coding.
- Deep Research, video generation, Google Flow, Gemini in Search/Gmail/Docs/Vids, NotebookLM higher access, Google Antigravity entry limits, Gemini in Android Studio, Google AI Studio higher limits, Google Flow Music.

Advice given:

- Good promo if the user will use Gemini app, NotebookLM, Google Workspace, AI Studio, or storage.
- Less direct if the goal is to power Hermes, because Google app subscriptions do not necessarily provide Hermes-usable API credit.

## OpenRouter budget framing

Advice given:

- $10 OpenRouter credit can last a month if the user uses cheap models for routine work and saves strong models for hard tasks.
- Recommended strategy:
  - cheap/default models for daily chat/study/light coding,
  - strong models for hard debugging, DevOps, repo analysis, architecture, or deep explanations.
- Agent/tool-heavy sessions burn more because file contents, tool outputs, and long context are included.

Example rough calculation used in the session:

- Cheap daily model around $0.15/M input and $0.60/M output: very low per normal task.
- Mid model around $1/M input and $2/M output: still many normal tasks per $10.
- Strong model around $3/M input and $15/M output: can be consumed quickly by heavy agent work.

Do not reuse these as exact current prices; use them as an explanatory pattern and verify live pricing.

## Student/free options surfaced

- Check **GitHub Student Developer Pack** first for a CS student.
- GitHub Copilot pricing page showed a free tier with limited monthly agent/chat requests and completions, and a student path for verified students.
- Perplexity student page existed but details were not verified in-session.
- Claude Pro was observed around $20/month, usually outside a strict $10 budget unless discounted.
- ChatGPT Pro promo was observed for Codex usage but was too expensive for the user's $10 target.

## Pitfalls

- Do not imply Google AI Pro automatically grants Hermes-ready API credit. Verify.
- Do not treat storage/workspace perks as valuable if the user says they only care about AI.
- Do not recommend a single expensive app subscription when a mixed strategy — free student tools + small API credit — would better fit the budget.
