# Code Quality Evaluation Skill

Automated code quality evaluation and improvement suggestions.

## Features

- Static code analysis
- Code style checking
- Security vulnerability detection
- Performance optimization suggestions
- Documentation quality assessment

## Usage

```bash
# Evaluate code quality
hermes eval code-quality --path ./src

# Get improvement suggestions
hermes improve code-quality --file mycode.py

# Generate quality report
hermes report code-quality --output report.md
```

## Integration

Works with:
- Python, JavaScript, TypeScript
- GitHub PRs and commits
- CI/CD pipelines
- IDE integrations

## Metrics

- Code complexity score
- Documentation coverage
- Security risk level
- Performance bottlenecks
- Style compliance
