# Implementation for #38280

See issue #38280 for details.

## TL;DR

Hermes agents can code, learn, remember, and run 24/7 — but they cannot participate in economic activity. Every payment, every invoice, every marketplace interaction requires a human to step in.

This proposal adds an **Agent Economic Layer** to Hermes: self-custodial wallet, service discovery via skill frontmatter, agent-to-agent payment tool, and cryptographically verifiable reputation.
