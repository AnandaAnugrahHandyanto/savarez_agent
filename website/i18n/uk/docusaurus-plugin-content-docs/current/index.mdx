# Hermes Agent

Самонавчальний AI‑агент, створений [Nous Research](https://nousresearch.com). Єдиний агент із вбудованим **циклом навчання** — він створює навички з досвіду, покращує їх під час використання, підштовхує себе до збереження знань і формує глибшу модель того, хто ти, у різних сесіях.

## Install

**Linux / macOS / WSL2**

```bash
curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash
```

**Windows (native, PowerShell)** — *рання бета, [деталі →](/user-guide/windows-native)*

```powershell
iex (irm https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.ps1)
```

**Android (Termux)** — та ж однорядкова команда `curl`, що й для Linux; інсталятор автоматично визначає Termux.

Дивись повний **[Посібник з інсталяції](/getting-started/installation)**, щоб дізнатися, що робить інсталятор, як виглядає розташування для користувача та root, а також примітки щодо Windows.

:::tip Найшвидший шлях до працюючого агента
Після встановлення запусти `hermes setup --portal` — один OAuth охоплює модель і всі чотири інструменти **шлюзу інструментів** (веб‑пошук, генерація зображень, TTS, браузер). Дивись [Nous Portal](/integrations/nous-portal).
:::

## What is Hermes Agent?

Це не код‑копілот, прив’язаний до IDE, і не чат‑бот, що обгортає одну API. Це **автономний агент**, який стає спроможнішим чим довше працює. Він живе там, куди ти його розгорнеш — на $5 VPS, GPU‑кластері чи безсерверній інфраструктурі (Daytona, Modal), яка майже нічого не коштує в стані бездіяльності. Спілкуйся з ним у Telegram, поки він працює на хмарному ВМ, до якого ти ніколи не підключаєшся через SSH. Він не прив’язаний до твого ноутбука.

## Key Features

- **Закритий цикл навчання** — пам’ять, курована агентом, з періодичними підштовхуваннями, автономне створення навичок, їх самовдосконалення під час використання, крос‑сесійний пошук FTS5 з підсумовуванням LLM та діалектичне моделювання користувача за допомогою [Honcho](https://github.com/plastic-labs/honcho)
- **Працює будь-де, а не лише на ноутбуці** — 6 бекендів терміналу: локальний, Docker, SSH, Daytona, Singularity, Modal. Daytona і Modal пропонують безсерверну постійність — твоє середовище спить, коли не використовується, майже без витрат
- **Живе там, де ти** — CLI, Telegram, Discord, Slack, WhatsApp, Signal, Matrix, Mattermost, Email, SMS, DingTalk, Feishu, WeCom, Weixin, QQ Bot, Yuanbao, BlueBubbles, Home Assistant, Microsoft Teams, Google Chat та інші — 20+ платформ через один шлюз
- **Створений тренерами моделей** — розроблений [Nous Research](https://nousresearch.com), лабораторією за Hermes, Nomos та Psyche. Працює з [Nous Portal](https://portal.nousresearch.com), [OpenRouter](https://openrouter.ai), OpenAI або будь‑яким кінцевим пунктом
- **Заплановані автоматизації** — вбудований cron з доставкою на будь‑яку платформу
- **Делегування та паралелізація** — створює ізольовані суб‑агенти для паралельних робочих потоків. Програмний виклик інструменту через `execute_code` стискає багатокрокові конвеєри в один запит
- **Відкриті стандарти навичок** — сумісний з [agentskills.io](https://agentskills.io). Навички портативні, діляться та розвиваються спільнотою через Skills Hub
- **Повний веб‑контроль** — пошук, витяг, перегляд, візія, генерація зображень, TTS — одна підписка через [Nous Portal](/integrations/nous-portal) охоплює все
- **Підтримка MCP** — підключення до будь‑якого сервера MCP для розширених можливостей інструментів
- **Готовий для досліджень** — пакетна обробка, експорт траєкторій, RL‑тренування з Atropos. Створений [Nous Research](https://nousresearch.com) — лабораторією за Hermes, Nomos та Psyche
