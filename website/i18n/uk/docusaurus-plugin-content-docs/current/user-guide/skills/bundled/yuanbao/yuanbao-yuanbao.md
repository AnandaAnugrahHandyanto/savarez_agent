## Робочий процес @згадки

Коли потрібно @згадати / 艾特 когось:

1. Викликати `yb_query_group_members` з `action="find"`, `name="<target name>"`, `mention=true`
2. Отримати точний нік з відповіді
3. Включити `@nickname` у текст відповіді — шлюз обробить решту

Приклад: користувач каже «帮我艾特元宝»

**Крок 1 — виклик інструмента:**
```json
{ "group_code": "328306697", "action": "find", "name": "元宝", "mention": true }
```

**Крок 2 — твоя відповідь** (це буде надіслано в групу з працюючою @згадкою):
```
@元宝 你好，有人找你！
```

**Ось і все.** Ніяких додаткових пояснень. Будь коротким і природним.

**Правила:**
- Спочатку виклич `yb_query_group_members`, щоб отримати точний нік — НЕ вгадуй.
- Формат @згадки: `@nickname` з пробілом перед знаком `@`.
- Текст твоєї відповіді — це повідомлення, воно БУДЕ надіслано, і @згадка ПРАЦЮЄ.
- Будь лаконічним. НЕ пояснюй користувачу, як працює @згадка.

## Робочий процес надсилання DM (приватного повідомлення)

Коли хтось просить надіслати приватне повідомлення / 私信 / DM користувачу:

1. Викликати `yb_send_dm` з параметрами `group_code`, `name` (ім’я цільового користувача) та `message`
2. Інструмент автоматично знаходить користувача і надсилає DM
3. Повідомити результат користувачу

**Приклад:** користувач каже «给 @用户aea3 私信发一个 hello»

```json
yb_send_dm({ "group_code": "535168412", "name": "用户aea3", "message": "hello" })
```

**Приклад з медіа:** користувач каже «给 @用户aea3 私信发一张图片»

```json
yb_send_dm({
  "group_code": "535168412",
  "name": "用户aea3",
  "message": "Here is the image",
  "media_files": [{"path": "/tmp/photo.jpg"}]
})
```

**Правила:**
- Витягнути `group_code` з поточного `chat_id` (наприклад, `group:535168412` → `535168412`).
- Якщо вже знаєш `user_id`, передай його через параметр `user_id`, щоб пропустити пошук.
- Якщо за ім’ям підходить кілька користувачів, інструмент поверне кандидатів — запитай уточнення.
- НЕ використовуйте інструмент `send_message` для DM у Юанбао — використовуйте `yb_send_dm`.
- Підтримуються медіа: зображення (.jpg/.png/.gif/.webp/.bmp) надсилаються як image‑повідомлення, інші файли — як документи.
