# Телефонія — номери, дзвінки та повідомлення без змін у Core‑інструментах

Цей необов’язковий **skill** надає Hermes практичні можливості телефонії, залишаючи її поза основним списком інструментів.

Він постачається з допоміжним скриптом `scripts/telephony.py`, який може:
- зберігати облікові дані провайдера у `~/.hermes/.env`
- шукати та купувати номер телефону Twilio
- запам’ятовувати придбаний номер для подальших сесій
- надсилати SMS/MMS з придбаного номера
- опитувати вхідні SMS для цього номера без необхідності веб‑хук сервера
- здійснювати прямі дзвінки Twilio, використовуючи TwiML `<Say>` або `<Play>`
- імпортувати придбаний номер Twilio у Vapi
- здійснювати вихідні AI‑дзвінки через Bland.ai або Vapi
## Що це вирішує

Цей інструмент призначений для покриття практичних телефонних завдань, які користувачі дійсно хочуть:
- вихідні дзвінки
- надсилання SMS
- володіння багаторазовим номером агента
- перевірка повідомлень, які надходять на цей номер пізніше
- збереження цього номера та пов’язаних ідентифікаторів між сесіями
- телефонна ідентичність, орієнтована на майбутнє, для вхідного SMS‑поллінгу та інших автоматизацій

Він **не** перетворює Hermes у шлюз реального часу для вхідних дзвінків. Вхідні SMS обробляються шляхом поллінгу Twilio REST API. Це достатньо для багатьох робочих процесів, включаючи сповіщення та отримання одноразових кодів, без додавання основної інфраструктури веб‑хуків.
## Правила безпеки — обов’язкові

1. Завжди підтверджуй дію перед здійсненням дзвінка або надсиланням повідомлення.
2. Ніколи не набирай номери екстрених служб.
3. Не використовуйте телефонію для переслідувань, спаму, видавання себе за іншу особу чи будь‑якої незаконної діяльності.
4. Считай сторонні телефонні номери конфіденційними операційними даними:
   - не зберігай їх у пам’яті Hermes;
   - не включай їх у документи навичок, резюме чи нотатки після дії, якщо користувач явно не вимагає цього.
5. Дозволено зберігати **agent-owned Twilio number**, оскільки це частина конфігурації користувача.
6. Номери VoIP **не гарантуються** для роботи у всіх сторонніх процесах 2FA. Використовуй їх обережно і чітко формулюй очікування користувача.
## Decision tree — який сервіс використовувати?

Використовуй цю логіку замість жорстко закодованого маршрутування провайдерів:

### 1) «Я хочу, щоб Hermes мав власний реальний номер телефону»
Використовуй **Twilio**.

**Чому:**
- найпростіший шлях до придбання та збереження номера
- найкраща підтримка SMS / MMS
- найпростіша реалізація опитування вхідних SMS
- найчистіший майбутній шлях до вхідних вебхук‑ів або обробки дзвінків

**Випадки використання:**
- отримувати текстові повідомлення пізніше
- надсилати сповіщення про розгортання / повідомлення cron
- підтримувати повторно використовувану телефонну ідентичність для агента
- експериментувати з телефонними потоками автентифікації в майбутньому

### 2) «Мені зараз потрібен лише найпростіший вихідний AI‑дзвінок»
Використовуй **Bland.ai**.

**Чому:**
- найшвидше налаштування
- один API‑ключ
- не потрібно спочатку купувати/імпортувати номер самостійно

**Компроміс:**
- менша гнучкість
- якість голосу задовільна, але не найкраща

### 3) «Я хочу найкращу якість голосу для розмовного AI»
Використовуй **Twilio + Vapi**.

**Чому:**
- Twilio надає власний номер
- Vapi забезпечує кращу якість розмовних AI‑дзвінків та більшу гнучкість голосових моделей

**Рекомендований процес:**
1. Придбати/зберегти номер Twilio
2. Імпортувати його в Vapi
3. Зберегти отриманий `VAPI_PHONE_NUMBER_ID`
4. Використовувати `ai-call --provider vapi`

### 4) «Я хочу здійснити дзвінок із власним заздалегідь записаним голосовим повідомленням»
Використовуй **Twilio direct call** з публічним URL аудіо‑файлу.

**Чому:**
- найпростіший спосіб відтворити власний MP3
- добре поєднується з Hermes `text_to_speech` та публічним файловим хостингом або тунелем
## Файли та постійний стан

Навик зберігає стан телефонії у двох місцях:

### `~/.hermes/.env`
Використовується для довготривалих облікових даних провайдера та ідентифікаторів власних номерів, наприклад:
- `TWILIO_ACCOUNT_SID`
- `TWILIO_AUTH_TOKEN`
- `TWILIO_PHONE_NUMBER`
- `TWILIO_PHONE_NUMBER_SID`
- `BLAND_API_KEY`
- `VAPI_API_KEY`
- `VAPI_PHONE_NUMBER_ID`
- `PHONE_PROVIDER` (AI‑call провайдер: bland або vapi)

### `~/.hermes/telephony_state.json`
Використовується для стану лише навика, який має зберігатися між сесіями, наприклад:
- запам’ятований типовий номер Twilio / SID
- запам’ятований ідентифікатор номеру Vapi
- останній вхідний SID/дата повідомлення для контрольних точок опитування вхідних повідомлень

Це означає:
- наступного разу, коли навик завантажується, `diagnose` може підказати, який номер вже налаштовано
- `twilio‑inbox --since-last --mark-seen` може продовжити з попередньої контрольної точки

## Налаштування провайдера

### Twilio — власний номер, SMS/MMS, прямі виклики, поллінг вхідних SMS

Зареєструйся за адресою:
- https://www.twilio.com/try-twilio

Потім збережи облікові дані в Hermes:

```bash
python3 "$SCRIPT" save-twilio ACXXXXXXXXXXXXXXXXXXXXXXXXXXXX your_auth_token_here
```

Пошук доступних номерів:

```bash
python3 "$SCRIPT" twilio-search --country US --area-code 702 --limit 5
```

Придбай і запам’ятай номер:

```bash
python3 "$SCRIPT" twilio-buy "+17025551234" --save-env
```

Список власних номерів:

```bash
python3 "$SCRIPT" twilio-owned
```

Пізніше встанови один із них як типово:

```bash
python3 "$SCRIPT" twilio-set-default "+17025551234" --save-env
# or
python3 "$SCRIPT" twilio-set-default PNXXXXXXXXXXXXXXXXXXXXXXXXXXXX --save-env
```

### Bland.ai — найпростіший вихідний AI‑виклик

Зареєструйся за адресою:
- https://app.bland.ai

Збережи конфігурацію:

```bash
python3 "$SCRIPT" save-bland your_bland_api_key --voice mason
```

### Vapi — краща якість розмовного голосу

Зареєструйся за адресою:
- https://dashboard.vapi.ai

Спочатку збережи API‑ключ:

```bash
python3 "$SCRIPT" save-vapi your_vapi_api_key
```

Імпортуй свій власний номер Twilio у Vapi і запам’ятай отриманий ідентифікатор телефонного номера:

```bash
python3 "$SCRIPT" vapi-import-twilio --save-env
```

Якщо ти вже знаєш ідентифікатор телефонного номера Vapi, збережи його безпосередньо:

```bash
python3 "$SCRIPT" save-vapi your_vapi_api_key --phone-number-id vapi_phone_number_id_here
```

## Загальні робочі процеси

### A. Купити номер агента і використовувати його пізніше

1. Збережи облікові дані Twilio:
```bash
python3 "$SCRIPT" save-twilio AC... auth_token_here
```

2. Пошукай номер:
```bash
python3 "$SCRIPT" twilio-search --country US --area-code 702 --limit 10
```

3. Купи його і збережи у `~/.hermes/.env` + стан:
```bash
python3 "$SCRIPT" twilio-buy "+17025551234" --save-env
```

4. У наступній сесії виконай:
```bash
python3 "$SCRIPT" diagnose
```
Це показує запам’ятований номер за замовчуванням і стан контрольної точки вхідних повідомлень.

### B. Надіслати текст з номера агента

```bash
python3 "$SCRIPT" twilio-send-sms "+15551230000" "Your deployment completed successfully."
```

З медіа:

```bash
python3 "$SCRIPT" twilio-send-sms "+15551230000" "Here is the chart." --media-url "https://example.com/chart.png"
```

### C. Перевірити вхідні тексти пізніше без сервера вебхука

Опитати вхідну скриньку для номера Twilio за замовчуванням:

```bash
python3 "$SCRIPT" twilio-inbox --limit 20
```

Показувати лише повідомлення, які надійшли після останньої контрольної точки, і просувати контрольну точку, коли завершиш читання:

```bash
python3 "$SCRIPT" twilio-inbox --since-last --mark-seen
```

Це основна відповідь на питання «як отримати доступ до повідомлень, які номер отримав наступного разу, коли навичка завантажується?»

### D. Здійснити прямий виклик Twilio з вбудованим TTS

```bash
python3 "$SCRIPT" twilio-call "+15551230000" --message "Hello! This is Hermes calling with your status update." --voice Polly.Joanna
```

### E. Виклик з заздалегідь записаним / кастомним голосовим повідомленням

Це основний шлях повторного використання існуючої підтримки `text_to_speech` у Hermes.

Використовуй це, коли:
- ти хочеш, щоб виклик використовував налаштований голос TTS Hermes, а не Twilio `<Say>`;
- потрібна однонаправлена голосова доставка (брифінг, тривога, жарт, нагадування, оновлення статусу);
- тобі **не** потрібен живий розмовний телефонний виклик.

Згенеруй або розмісти аудіо окремо, потім:

```bash
python3 "$SCRIPT" twilio-call "+155****0000" --audio-url "https://example.com/briefing.mp3"
```

Рекомендований робочий процес Hermes TTS → Twilio Play:

1. Згенеруй аудіо за допомогою Hermes `text_to_speech`.
2. Зроби отриманий MP3 публічно доступним.
3. Виконай виклик Twilio з `--audio-url`.

Приклад потоку агента:
- Попроси Hermes створити аудіо‑повідомлення за допомогою `text_to_speech`;
- За потреби, опублікуй файл через тимчасовий статичний хост / тунель / URL сховища об’єктів;
- Використай `twilio-call --audio-url …` для доставки його телефоном.

Хороші варіанти хостингу MP3:
- тимчасовий публічний URL об’єкта/сховища;
- короткоживучий тунель до локального статичного файлового сервера;
- будь‑який існуючий HTTPS‑URL, який телефонний провайдер може отримати безпосередньо.

**Важлива примітка**
- Hermes TTS підходить для заздалегідь записаних вихідних повідомлень.
- Bland/Vapi краще підходять для **живих розмовних AI‑викликів**, оскільки вони самостійно обробляють стек аудіо в реальному часі.
- Hermes STT/TTS сам по собі не використовується тут як повноцінний двосторонній телефонний рушій; це вимагало б набагато важчої інтеграції потокового/вебхук‑взаємодії, ніж передбачає ця навичка.

### F. Навігація по телефонному дереву / IVR за допомогою прямого виклику Twilio

Якщо потрібно натискати цифри після підключення виклику, використай `--send-digits`.
Twilio інтерпретує `w` як коротку паузу.

```bash
python3 "$SCRIPT" twilio-call "+18005551234" --message "Connecting to billing now." --send-digits "ww1w2w3"
```

Це корисно для переходу до конкретної гілки меню перед передачею виклику людині або доставкою короткого статусного повідомлення.

### G. Вихідний AI‑виклик телефоном з Bland.ai

```bash
python3 "$SCRIPT" ai-call "+15551230000" "Call the dental office, ask for a cleaning appointment on Tuesday afternoon, and if they do not have Tuesday availability, ask for Wednesday or Thursday instead." --provider bland --voice mason --max-duration 3
```

Перевір статус:

```bash
python3 "$SCRIPT" ai-status <call_id> --provider bland
```

Задай питання аналізу Bland після завершення:

```bash
python3 "$SCRIPT" ai-status <call_id> --provider bland --analyze "Was the appointment confirmed?,What date and time?,Any special instructions?"
```

### H. Вихідний AI‑виклик телефоном з Vapi на твоєму власному номері

1. Імпортуй свій номер Twilio у Vapi:
```bash
python3 "$SCRIPT" vapi-import-twilio --save-env
```

2. Здійсни виклик:
```bash
python3 "$SCRIPT" ai-call "+15551230000" "You are calling to make a dinner reservation for two at 7:30 PM. If that is unavailable, ask for the nearest time between 6:30 and 8:30 PM." --provider vapi --max-duration 4
```

3. Перевір результат:
```bash
python3 "$SCRIPT" ai-status <call_id> --provider vapi
```
## Запропонована процедура агента

Коли користувач запитує про дзвінок або повідомлення:

1. Визначити, який шлях відповідає запиту, за допомогою дерева рішень.
2. Запустити `diagnose`, якщо стан конфігурації незрозумілий.
3. Зібрати повну інформацію про завдання.
4. Підтвердити дію з користувачем перед набором номера або надсиланням повідомлення.
5. Використати правильну команду.
6. За потреби опитати результат.
7. Підсумувати результат, не зберігаючи сторонні номери у пам'ять Hermes.
## Що цей інструмент ще не робить

- відповіді на вхідні дзвінки в режимі реального часу
- надсилання SMS у режимі реального часу за допомогою webhook у цикл агента
- гарантована підтримка довільних сторонніх постачальників 2FA

Для цього знадобиться більше інфраструктури, ніж у чисто необов’язковому інструменті.
## Підводні камені

- Twilio пробні облікові записи та регіональні правила можуть обмежувати, кому можна дзвонити/надсилати повідомлення.
- Деякі сервіси відхиляють VoIP‑номери для 2FA.
- `twilio‑inbox` опитує REST API; це не миттєва push‑доставка.
- Vapi outbound calling все ще залежить від наявності дійсного імпортованого номера.
- Bland найпростіший, але не завжди найкращий за звучанням.
- Не зберігай довільні сторонні телефонні номери в пам'яті Hermes.

4. надіслати SMS з придбаного номера
5. опитати вхідні повідомлення для придбаного номера пізніше
6. здійснити прямий виклик через Twilio
7. здійснити AI‑виклик через Bland або Vapi
