# Модель злиття

Будуй аналіз акреції/дилюції для M&A‑транзакцій. Моделі про‑форму впливу на EPS, чутливість синергій та розподіл вартості покупки. Використовуй при оцінці потенційного придбання, підготовці аналізу наслідків злиття для презентації або консультуванні щодо умов угоди.

## Робочий процес

### Крок 1: Збір вхідних даних

**Придбувач:**
- Назва компанії, поточна ціна акції, кількість акцій в обігу
- LTM та NTM EPS (GAAP та скоригований)
- P/E множник
- Передподаткові витрати на борг, податкова ставка
- Готівка на балансі, існуючий борг

**Ціль:**
- Назва компанії, поточна ціна акції, кількість акцій в обігу (якщо публічна)
- LTM та NTM EPS або чистий прибуток
- Вартість підприємства або вартість капіталу

**Умови угоди:**
- Ціна пропозиції за акцію (або премія до поточної)
- Склад частки розгляду: % готівки vs. % акцій
- Новий борг, залучений для фінансування готівкової частини
- Очікувані синергії (дохід та витрати) та графік їх впровадження
- Транзакційні збори та витрати на фінансування
- Очікувана дата закриття

## Важливі нотатки

- Завжди показуй як GAAP, так і скоригований (cash) EPS, де це релевантно
- Угоди з акціями: використай поточну ціну придбувача для коефіцієнту обміну, зазначай дилюцію від нових акцій
- Включай розподіл вартості покупки — goodwill та амортизація нематеріальних активів впливають на GAAP EPS
- Фаза впровадження синергій критична — у Рік 1 часто лише 25‑50 % від повної швидкості синергій
- Не забудь про втрачені доходи від відсотків на готівці та нові витрати на відсотки по залученому боргу
- Податкова ставка на синергії та корекції відсотків має відповідати маржинальній ставці придбувача
