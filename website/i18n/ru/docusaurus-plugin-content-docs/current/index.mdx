# Hermes Agent

Самообучающийся AI‑агент, созданный [Nous Research](https://nousresearch.com). Единственный агент с встроенным циклом обучения — он создаёт навыки из опыта, улучшает их во время использования, подталкивает себя к сохранению знаний и формирует углублённую модель того, кто ты, сквозь сессии.

## Install

**Linux / macOS / WSL2**

```bash
curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash
```

**Windows (native, PowerShell)** — *ранняя бета, [details →](/user-guide/windows-native)*

```powershell
iex (irm https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.ps1)
```

**Android (Termux)** — тот же однострочный `curl`, что и для Linux; установщик автоматически определяет Termux.

Смотрите полный **[Installation Guide](/getting-started/installation)**, чтобы узнать, что делает установщик, различия между пользовательской и root‑структурой и примечания для Windows.

:::tip Fastest path to a working agent
После установки выполните `hermes setup --portal` — один OAuth покрывает модель и все четыре инструмента шлюза (веб‑поиск, генерация изображений, TTS, браузер). См. [Nous Portal](/integrations/nous-portal).
:::

## What is Hermes Agent?

Это не код‑копилот, привязанный к IDE, и не чат‑бот‑обёртка вокруг одного API. Это **автономный агент**, который становится способнее, чем дольше работает. Он живёт там, где ты его разместишь — на VPS за $5, в GPU‑кластере или в безсерверной инфраструктуре (Daytona, Modal), которая почти ничего не стоит в простое. Общайся с ним через Telegram, пока он работает на облачном VM, к которому ты сам никогда не подключаешься по SSH. Он не привязан к твоему ноутбуку.

## Key Features

- **Закрытый цикл обучения** — память, курируемая агентом, с периодическими подталкиваниями, автономное создание навыков, их самоулучшение в процессе использования, кросс‑сессионный поиск FTS5 с суммированием LLM и диалектическое моделирование пользователя от [Honcho](https://github.com/plastic-labs/honcho)
- **Работает где угодно, а не только на ноутбуке** — 6 бекендов терминала: локальный, Docker, SSH, Daytona, Singularity, Modal. Daytona и Modal предоставляют безсерверную постоянность — среда спит в простое, почти не тратя ресурсы
- **Живёт там, где живёшь ты** — CLI, Telegram, Discord, Slack, WhatsApp, Signal, Matrix, Mattermost, Email, SMS, DingTalk, Feishu, WeCom, Weixin, QQ Bot, Yuanbao, BlueBubbles, Home Assistant, Microsoft Teams, Google Chat и др. — более 20 платформ через один шлюз
- **Создан тренерами моделей** — разработан [Nous Research](https://nousresearch.com), лабораторией, стоящей за Hermes, Nomos и Psyche. Работает с [Nous Portal](https://portal.nousresearch.com), [OpenRouter](https://openrouter.ai), OpenAI и любым другим эндпоинтом
- **Запланированные автоматизации** — встроенный cron с доставкой на любую платформу
- **Делегирует и параллелит** — порождает изолированные суб‑агенты для параллельных потоков работы. Программный вызов инструментов через `execute_code` сворачивает многошаговые конвейеры в один инференс‑запрос
- **Открытые стандарты навыков** — совместимость с [agentskills.io](https://agentskills.io). Навыки портативны, делятся и поддерживаются сообществом через Skills Hub
- **Полный веб‑контроль** — поиск, извлечение, браузинг, зрение, генерация изображений, TTS — одна подписка через [Nous Portal](/integrations/nous-portal) объединяет всё
- **Поддержка MCP** — подключение к любому MCP‑серверу для расширения возможностей инструментов
- **Готов к исследованиям** — пакетная обработка, экспорт траекторий, RL‑тренировка с Atropos. Разработан [Nous Research](https://nousresearch.com) — лабораторией, стоящей за моделями Hermes, Nomos и Psyche

## For LLMs and coding agents

- **[`/llms.txt`](/llms.txt)** — отобранный индекс всех страниц с короткими описаниями. ~17 KB, безопасно загрузить в контекст LLM.
