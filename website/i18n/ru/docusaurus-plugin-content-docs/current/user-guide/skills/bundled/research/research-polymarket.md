# Polymarket — данные предсказательных рынков

Запрашивай данные предсказательных рынков из Polymarket с помощью их публичных REST API.
Все эндпоинты только для чтения и не требуют аутентификации.

См. `references/api-endpoints.md` для полного справочника эндпоинтов с примерами curl.

## Когда использовать

- Пользователь спрашивает о предсказательных рынках, коэффициентах ставок или вероятностях событий
- Пользователь хочет узнать «каковы шансы того, что X произойдёт?»
- Пользователь интересуется конкретно Polymarket
- Пользователь хочет цены рынка, данные книги ордеров или историю цен
- Пользователь просит мониторить или отслеживать движения предсказательного рынка

## Ключевые концепции

- **Events** содержат один или несколько **Markets** (отношение 1:много)
- **Markets** — бинарные исходы с ценами Yes/No от 0.00 до 1.00
- Цены ЯВЛЯЮТСЯ вероятностями: цена 0.65 означает, что рынок считает вероятность 65 %
- Поле `outcomePrices`: JSON‑массив, например `["0.80", "0.20"]`
- Поле `clobTokenIds`: JSON‑массив из двух токен‑идентификаторов [Yes, No] для запросов цены/книги
- Поле `conditionId`: hex‑строка, используемая для запросов истории цен
- Объём измеряется в USDC (доллары США)

## Три публичных API

1. **Gamma API** на `gamma-api.polymarket.com` — обнаружение, поиск, навигация
2. **CLOB API** на `clob.polymarket.com` — цены в реальном времени, книги ордеров, история
3. **Data API** на `data-api.polymarket.com` — сделки, открытый интерес

## Типичный рабочий процесс

Когда пользователь спрашивает о коэффициентах предсказательного рынка:

1. **Поиск** с помощью публичного эндпоинта Gamma API `public-search`, передавая запрос пользователя
2. **Разбор** ответа — извлечь события и вложенные в них рынки
3. **Представление** вопроса рынка, текущих цен в процентах и объёма
4. **Углублённый анализ** по запросу — использовать `clobTokenIds` для книги ордеров, `conditionId` для истории

## Представление результатов

Форматируй цены в процентах для удобства чтения:
- `outcomePrices` `["0.652", "0.348"]` превращается в «Yes: 65.2 %, No: 34.8 %»
- Всегда показывай вопрос рынка и вероятность
- Добавляй объём, если он доступен

Пример: `"Will X happen?" — 65.2 % Yes ($1.2M volume)`

## Парсинг двойных кодированных полей

Gamma API возвращает `outcomePrices`, `outcomes` и `clobTokenIds` как JSON‑строки внутри JSON‑ответов (двойное кодирование). При обработке в Python разбирай их через
`json.loads(market['outcomePrices'])`, чтобы получить реальный массив.

## Ограничения по частоте запросов

Щедрые — маловероятно превысить при обычном использовании:
- Gamma: 4 000 запросов за 10 секунд (общие)
- CLOB: 9 000 запросов за 10 секунд (общие)
- Data: 1 000 запросов за 10 секунд (общие)

## Ограничения

- Этот навык только для чтения — он не поддерживает размещение сделок
- Торговля требует криптовалютной аутентификации через кошелёк (подписи EIP-712)
- У некоторых новых рынков может отсутствовать история цен
- На торговлю действуют географические ограничения, но данные только для чтения доступны глобально
