# Телефония — номера, звонки и сообщения без изменений основного набора инструментов

Этот необязательный навык предоставляет Hermes практические возможности работы с телефоном, оставляя телефонию вне списка основных инструментов.

Он поставляется с вспомогательным скриптом `scripts/telephony.py`, который может:
- сохранять учётные данные провайдера в `~/.hermes/.env`
- искать и покупать номер телефона Twilio
- запоминать этот номер для последующих сессий
- отправлять SMS/MMS с этого номера
- получать входящие SMS для этого номера без необходимости сервера веб‑хуков
- осуществлять прямые звонки Twilio с использованием TwiML `<Say>` или `<Play>`
- импортировать этот номер Twilio в Vapi
- делать исходящие AI‑звонки через Bland.ai или Vapi
## Что это решает

Этот навык предназначен для выполнения практических телефонных задач, которые действительно нужны пользователям:
- исходящие звонки
- отправка SMS
- владение переиспользуемым номером агента
- проверка сообщений, приходящих на этот номер позже
- сохранение этого номера и связанных идентификаторов между сессиями
- будущая совместимая телефонная идентификация для входящего опроса SMS и других автоматизаций

Он **не** превращает Hermes в шлюз входящих звонков в реальном времени. Входящие SMS обрабатываются путём опроса Twilio REST API. Это достаточно для многих рабочих процессов, включая уведомления и получение одноразовых кодов, без добавления основной инфраструктуры веб‑хуков.
## Правила безопасности — обязательные

1. Всегда подтверждай действие перед совершением звонка или отправкой сообщения.
2. Никогда не набирай экстренные номера.
3. Никогда не используй телефонную связь для преследования, спама, выдачи себя за другого или любой незаконной деятельности.
4. Относись к телефонным номерам третьих сторон как к конфиденциальным оперативным данным:
   - не сохраняй их в памяти Hermes;
   - не включай их в документы навыков, резюме или последующие заметки, если только пользователь явно не запросит этого.
5. Можно сохранять **agent-owned Twilio number**, так как это часть конфигурации пользователя.
6. Номера VoIP **не гарантируют** работу со всеми сторонними процессами 2FA. Используй их с осторожностью и чётко формулируй ожидания пользователя.
## Дерево решений — какой сервис использовать?

Используй эту логику вместо жёстко заданной маршрутизации провайдеров:

### 1) «Я хочу, чтобы Hermes владел реальным номером телефона»
Используй **Twilio**.

**Почему:**
- самый простой путь к покупке и удержанию номера
- лучшая поддержка SMS/MMS
- самая простая схема опроса входящих SMS
- самый чистый будущий путь к входящим веб‑хукам или обработке звонков

**Сценарии использования:**
- получать сообщения позже
- отправлять оповещения о развертывании / уведомления cron
- поддерживать повторно используемую телефонную идентичность для агента
- экспериментировать с аутентификационными потоками на основе телефона позже

### 2) «Мне сейчас нужен самый простой исходящий AI‑звонок»
Используй **Bland.ai**.

**Почему:**
- самая быстрая настройка
- один API‑ключ
- не нужно сначала покупать / импортировать номер самостоятельно

**Компромисс:**
- менее гибко
- качество голоса приемлемое, но не лучшее

### 3) «Я хочу лучшее качество разговорного AI‑голоса»
Используй **Twilio + Vapi**.

**Почему:**
- Twilio даёт тебе собственный номер
- Vapi обеспечивает более высокое качество разговорного AI‑звонка и большую гибкость голосов/моделей

**Рекомендуемый поток:**
1. Купить / сохранить номер Twilio
2. Импортировать его в Vapi
3. Сохранить полученный `VAPI_PHONE_NUMBER_ID`
4. Использовать `ai-call --provider vapi`

### 4) «Я хочу звонить с пользовательским заранее записанным голосовым сообщением»
Используй **Twilio direct call** с публичным URL аудио.

**Почему:**
- самый простой способ воспроизвести пользовательский MP3
- хорошо сочетается с Hermes `text_to_speech` плюс публичный файловый хост или туннель
## Файлы и постоянное состояние

Навык сохраняет состояние телефонной связи в двух местах:

### `~/.hermes/.env`
Используется для долгосрочных учётных данных провайдера и идентификаторов принадлежащих номеров, например:
- `TWILIO_ACCOUNT_SID`
- `TWILIO_AUTH_TOKEN`
- `TWILIO_PHONE_NUMBER`
- `TWILIO_PHONE_NUMBER_SID`
- `BLAND_API_KEY`
- `VAPI_API_KEY`
- `VAPI_PHONE_NUMBER_ID`
- `PHONE_PROVIDER` (AI‑провайдер звонков: bland или vapi)

### `~/.hermes/telephony_state.json`
Используется для состояния, относящегося только к навыку, которое должно сохраняться между сессиями, например:
- запомненный номер Twilio по умолчанию / SID
- запомненный идентификатор телефонного номера Vapi
- последний входящий SID сообщения/дата для контрольных точек опроса входящих сообщений

Это означает:
- при следующей загрузке навыка `diagnose` сможет сообщить, какой номер уже сконфигурирован
- `twilio‑inbox --since-last --mark-seen` может продолжить работу с предыдущей контрольной точкой

## Настройка провайдера

### Twilio — собственный номер, SMS/MMS, прямые звонки, опрос входящих SMS

Зарегистрируйся на:
- https://www.twilio.com/try-twilio

Затем сохрани учётные данные в Hermes:

```bash
python3 "$SCRIPT" save-twilio ACXXXXXXXXXXXXXXXXXXXXXXXXXXXX your_auth_token_here
```

Найди доступные номера:

```bash
python3 "$SCRIPT" twilio-search --country US --area-code 702 --limit 5
```

Приобрети и запомни номер:

```bash
python3 "$SCRIPT" twilio-buy "+17025551234" --save-env
```

Получи список собственных номеров:

```bash
python3 "$SCRIPT" twilio-owned
```

Позже установи один из них как номер по умолчанию:

```bash
python3 "$SCRIPT" twilio-set-default "+17025551234" --save-env
# or
python3 "$SCRIPT" twilio-set-default PNXXXXXXXXXXXXXXXXXXXXXXXXXXXX --save-env
```

### Bland.ai — самый простой исходящий AI‑звонок

Зарегистрируйся на:
- https://app.bland.ai

Сохрани конфигурацию:

```bash
python3 "$SCRIPT" save-bland your_bland_api_key --voice mason
```

### Vapi — лучшее качество разговорного голоса

Зарегистрируйся на:
- https://dashboard.vapi.ai

Сначала сохрани API‑ключ:

```bash
python3 "$SCRIPT" save-vapi your_vapi_api_key
```

Импортируй свой номер Twilio в Vapi и сохрани возвращённый идентификатор телефонного номера:

```bash
python3 "$SCRIPT" vapi-import-twilio --save-env
```

Если уже знаешь идентификатор телефонного номера Vapi, сохрани его напрямую:

```bash
python3 "$SCRIPT" save-vapi your_vapi_api_key --phone-number-id vapi_phone_number_id_here
```

## Общие рабочие процессы

### A. Купить номер агента и использовать его позже

1. Сохранить учётные данные Twilio:
```bash
python3 "$SCRIPT" save-twilio AC... auth_token_here
```

2. Найти номер:
```bash
python3 "$SCRIPT" twilio-search --country US --area-code 702 --limit 10
```

3. Купить его и сохранить в `~/.hermes/.env` + состояние:
```bash
python3 "$SCRIPT" twilio-buy "+17025551234" --save-env
```

4. В следующей сессии выполнить:
```bash
python3 "$SCRIPT" diagnose
```
Отобразится запомненный номер по умолчанию и состояние контрольной точки входящих сообщений.

### B. Отправить текст с номера агента

```bash
python3 "$SCRIPT" twilio-send-sms "+15551230000" "Your deployment completed successfully."
```

С медиа:

```bash
python3 "$SCRIPT" twilio-send-sms "+15551230000" "Here is the chart." --media-url "https://example.com/chart.png"
```

### C. Проверить входящие сообщения позже без сервера веб‑хуков

Опрашивать входящие сообщения для номера Twilio по умолчанию:

```bash
python3 "$SCRIPT" twilio-inbox --limit 20
```

Показывать только сообщения, пришедшие после последней контрольной точки, и продвигать контрольную точку после чтения:

```bash
python3 "$SCRIPT" twilio-inbox --since-last --mark-seen
```

Это основной ответ на вопрос «как получить доступ к сообщениям, полученным номером, при следующей загрузке навыка?»

### D. Совершить прямой звонок Twilio с встроенным TTS

```bash
python3 "$SCRIPT" twilio-call "+15551230000" --message "Hello! This is Hermes calling with your status update." --voice Polly.Joanna
```

### E. Звонок с заранее записанным/пользовательским голосовым сообщением

Это основной путь повторного использования поддержки `text_to_speech` в Hermes.

Используй его, когда:
- нужен звонок с настроенным голосом TTS Hermes, а не `<Say>` от Twilio;
- требуется однонаправленная голосовая доставка (брифинг, оповещение, шутка, напоминание, обновление статуса);
- **не** нужен живой разговорный телефонный звонок.

Сгенерируй или размести аудио отдельно, затем:

```bash
python3 "$SCRIPT" twilio-call "+155****0000" --audio-url "https://example.com/briefing.mp3"
```

Рекомендуемый рабочий процесс Hermes TTS → Twilio Play:

1. Сгенерировать аудио с помощью Hermes `text_to_speech`.
2. Сделать полученный MP3 общедоступным.
3. Совершить звонок Twilio с `--audio-url`.

Пример потока агента:
- Попросить Hermes создать аудио сообщения с `text_to_speech`;
- При необходимости открыть файл через временный статический хост / туннель / URL объектного хранилища;
- Использовать `twilio-call --audio-url ...` для доставки по телефону.

Хорошие варианты хостинга MP3:
- временный публичный URL объектного хранилища;
- короткоживущий туннель к локальному статическому файловому серверу;
- любой существующий HTTPS‑URL, который телефонный провайдер может загрузить напрямую.

Важно:
- Hermes TTS отлично подходит для заранее записанных исходящих сообщений;
- Bland/Vapi лучше для **живых разговорных AI‑звонков**, так как они сами обрабатывают стек аудио в реальном времени;
- Hermes STT/TTS здесь используется только для генерации аудио, а не как полноценный двусторонний телефонный разговорный движок — для этого потребовалась бы более тяжёлая интеграция потоков/веб‑хуков, чем предполагает данный навык.

### F. Навигация по телефонному меню / IVR с прямыми звонками Twilio

Если нужно нажимать цифры после подключения звонка, используй `--send-digits`. Twilio интерпретирует `w` как короткую паузу.

```bash
python3 "$SCRIPT" twilio-call "+18005551234" --message "Connecting to billing now." --send-digits "ww1w2w3"
```

Это полезно для доступа к конкретному пункту меню перед передачей человеку или отправкой короткого статусного сообщения.

### G. Исходящий AI‑звонок с Bland.ai

```bash
python3 "$SCRIPT" ai-call "+15551230000" "Call the dental office, ask for a cleaning appointment on Tuesday afternoon, and if they do not have Tuesday availability, ask for Wednesday or Thursday instead." --provider bland --voice mason --max-duration 3
```

Проверить статус:

```bash
python3 "$SCRIPT" ai-status <call_id> --provider bland
```

Задать вопросы анализа Bland после завершения:

```bash
python3 "$SCRIPT" ai-status <call_id> --provider bland --analyze "Was the appointment confirmed?,What date and time?,Any special instructions?"
```

### H. Исходящий AI‑звонок с Vapi на твоём собственном номере

1. Импортировать свой номер Twilio в Vapi:
```bash
python3 "$SCRIPT" vapi-import-twilio --save-env
```

2. Совершить звонок:
```bash
python3 "$SCRIPT" ai-call "+15551230000" "You are calling to make a dinner reservation for two at 7:30 PM. If that is unavailable, ask for the nearest time between 6:30 and 8:30 PM." --provider vapi --max-duration 4
```

3. Проверить результат:
```bash
python3 "$SCRIPT" ai-status <call_id> --provider vapi
```
## Предлагаемая процедура агента

Когда пользователь запрашивает звонок или сообщение:

1. Определить, какой путь соответствует запросу, используя дерево решений.
2. Выполнить `diagnose`, если состояние конфигурации неясно.
3. Собрать полные детали задания.
4. Подтвердить с пользователем действие перед набором номера или отправкой сообщения.
5. Использовать правильную команду.
6. При необходимости получать результаты.
7. Подвести итог, не сохраняя сторонние номера в памяти Hermes.
## Что этот skill всё ещё не умеет

- отвечать на входящие звонки в режиме реального времени
- отправлять SMS в реальном времени через webhook в цикл агента
- гарантированно поддерживать произвольных сторонних провайдеров 2FA

Для этого потребовалась бы более сложная инфраструктура, чем у чисто опционального skill.
## Подводные камни

- Тестовые аккаунты Twilio и региональные правила могут ограничивать, кому ты можешь звонить/отправлять сообщения.
- Некоторые сервисы отклоняют VoIP‑номера для 2FA.
- `twilio‑inbox` опрашивает REST API; это не мгновенная push‑доставка.
- Исходящие звонки Vapi всё ещё зависят от наличия действующего импортированного номера.
- Bland — самый простой, но не всегда звучит лучше.
- Не сохраняй произвольные сторонние телефонные номера в памяти Hermes.
